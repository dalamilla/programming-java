# Request Header Parser

Request Header Parser microservice implemented on Micronaut.

## Instructions

To start this app:

- Build Application:

```
mvn clean install
```

- Running Micronaut Application:

```
mvn mn:run
```

or

```
java -jar target/request-header-0.1.0.jar
```

- Run Micronaut Test Application:

```
mvn test
```
